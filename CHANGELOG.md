# Changelog

All notable changes to SaveMeGB are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Pro tier scaffolding (license validation, Pro-only features gated)
- 9 launchers supported: Steam, Epic, Xbox, GOG, Battle.net, Riot, EA, Ubisoft, manual installs
- 3 scan modes: quick, standard, deep
- Orphan detection across 7 categories: saves, cache, shaders, logs, crashes, settings, backups
- Library screen with Steam cover art
- Insights screen with SVG charts (donut + bar)
- Detail modal with real file tree
- Bulk select by category
- "Delete all visible" action
- Celebration modal with confetti
- Recycle Bin, Backup folder, and Force Delete strategies
- Export to JSON and CSV
- Whitelist for paths and game names
- First-run onboarding
- Keyboard shortcuts overlay (press `?`)
- Click-to-copy paths
- Click-to-open in Explorer

### Changed
- Renamed from SaveLock to SaveMeGB
- v0.1.0: First public release

## [0.1.0] - 2026-06-18

Initial open-source release.
